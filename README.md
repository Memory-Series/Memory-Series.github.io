# Memory Series

Memory Series 是一个围绕“数字记忆”与“数字生命”展开的品牌内容仓库。

这个仓库当前主要用于沉淀 Memory 的核心产品文档，包括品牌母文案、官网信息架构、首页文案草稿，以及五条产品线的产品页文案。整体表达从“记忆的留存”出发，延展到“记忆的重现、交互与持续存在”。

## 项目定位

Memory 希望建立一种更完整的记忆表达方式。

我们关注的，不只是内容被保存下来，而是人与记忆、人与情感、人与存在感之间的连接，如何通过对话、空间、角色、实体与日常记录被重新组织和延续。

一句话概括：

> Memory 是一个以数字记忆为起点、以数字生命为延展的品牌。

## 品牌原则

- 真实优先：不追求对人的机械复制，更重视语气、关系、行为逻辑与真实感。
- 克制表达：技术服务情感，而不是压过情感本身。
- 多维重构：从对话到空间、从角色到实体，让同一份记忆在不同维度中继续存在。
- 隐私与尊重：所有与记忆相关的数字化重构，都建立在边界、尊重与审慎之上。

## 产品矩阵

### Memory · Soulpod

以对话为入口，提取语言习惯、表达方式与人格特征，让记忆中的回应以更贴近真实气质的方式再次发生。

### Memory · Verse

以空间重建为基础，将图像、片段与环境线索转化为可进入的数字场域，让记忆中的场景可以被重新进入。

### Memory · Trace / Inhabit

以行为逻辑、角色能力与知识结构为核心，探索数字角色与数字生命更长期、更持续的存在方式。

### Memory · Sculpt

以图像转译、三维生成与实体输出为基础，让数字记忆获得可被看见、摆放与触摸的实体形态。

### Memory · FluffyDiary

以宠物第一人称日记为入口，将照片、视频与生活片段转化为可持续积累的纪念内容，连接当下陪伴与未来回看。
