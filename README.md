## 项目简介

### Trace · 寻迹

- 从小说、剧本、动漫素材提取角色人格
- 分析语言风格、性格特征、行为模式
- 输出 SoulPod（`profile.json` / `system_prompts.txt` / `memories`）

### Inhabit · 入心

- 加载 SoulPod，以角色身份对话
- 模式：复刻模式 / 伴侣模式 / 故事模式
- 能力：角色场景对话，角色图像生成，角色语音生成
